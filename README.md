### Sobre

Este é um pequeno projeto em React que criei para explorar o uso de componentes e props. O projeto consiste em um menu de pizza simples, onde cada pizza é representada por um componente individual. Utilizei esse projeto como uma maneira prática de compreender como as props podem ser usadas para passar dados entre os componentes e como criar componentes reutilizáveis para construir interfaces mais complexas.

Este projeto foi iniciado com Create React App.

### Scripts Disponíveis
No diretório do projeto, você pode executar:

### `npm start`
Executa o aplicativo no modo de desenvolvimento.
Abra http://localhost:3000 para visualizá-lo em seu navegador.

A página será recarregada quando você fizer alterações.
Você também poderá ver quaisquer erros de lint no console.
